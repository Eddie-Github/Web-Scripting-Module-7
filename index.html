JSON Objects

JSON can store objects containing multiple key value pairs. A JavaScript object can be created with a JSON object literal.

JSON Arrays

JSON can store arrays. A JavaScript array can be parsed from a JSON array.

JSON Null Values

JSON is capable of storing null values.
